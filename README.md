# Orisium

## Installation

The main development and testing environment currently uses **OpenBSD-7.8**.

```bash
git clone https://github.com/orichain/orisium.git
cd orisium
git submodule update --init --recursive
gmake clean debug
```

Installing neovim

```bash
./install-ori-nvim.sh
```
